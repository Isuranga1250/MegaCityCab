About the Project
The Mega City Cab system is a Java-based car rental service that allows customers to book rides, manage trips, and handle payments. The system includes features such as driver management, booking authentication, journey completion, and payment processing.


Technology Stack
Frontend: HTML, CSS, Bootstrap
Backend: Java (JSP, Servlets)
Database: Text File Storage (No SQL Database used)
Version Control: GitHub
